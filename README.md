# YARP Proxy Manager (YPM)

YARP Proxy Manager (YPM) is a modern, .NET-native reverse proxy management platform built to make secure host publishing straightforward. It gives you a clean way to route traffic, terminate SSL, and manage proxy behavior without wrestling with manual web server configuration.

Much of the inspiration comes from the great project [NGINX Proxy Manager](https://github.com/NginxProxyManager/nginx-proxy-manager).

Built on [YARP](https://learn.microsoft.com/en-us/aspnet/core/fundamentals/servers/yarp/getting-started?view=aspnetcore-10.0), YPM combines familiar proxy capabilities with an architecture that feels natural to .NET teams: composable, extensible, and portable across development and production environments.

## Features
- Easy setup of proxy hosts, powered by [YARP](https://learn.microsoft.com/en-us/aspnet/core/fundamentals/servers/yarp/getting-started?view=aspnetcore-10.0)
- Written entirely in .NET, making it easy for .NET developers to extend
- Portable across Windows, Linux, and macOS
- Advanced configuration options for power users
- User management and audit logging
- Custom SSL certificate management
